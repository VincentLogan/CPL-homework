二叉搜索树 (BST) 使用说明

本文件夹包含四个主要文件，用于完成实验逻辑开发与大规模数据测试。

1. 文件说明
  main.c        : 主程序代码（包含建树、遍历、查找等核心逻辑）。
  data_gen.c    : 测试数据生成器，用于自动生成大规模的平衡二叉树数据。
  test_data.txt : 存放测试数据的文本文件（可由 data_gen 生成或手动编辑）。
  result.txt: 存放了极端情况测试结果（在实验报告中详细描述）。


2. 编译步骤
在终端（CMD 或 PowerShell）中进入当前目录，使用 gcc 编译：

编译主程序：
gcc main.c -o main.exe

编译生成器：
gcc data_gen.c -o data_gen.exe


3. 使用大规模数据测试

    第一步：生成测试数据
    运行生成器并将输出保存到文件：
    ./data_gen.exe > test_data.txt

    第二步：运行主程序读取数据
    这一步需要根据你使用的终端类型选择不同的指令：

        情况 A: 如果你使用的是 CMD (命令提示符)
        直接使用标准的输入重定向符 `<`：
        main.exe < test_data.txt

        情况 B: 如果你使用的是 PowerShell (VS Code 默认终端)
        注意：PowerShell 不支持使用 `<` 进行输入重定向。请改用以下“管道”指令：
        Get-Content test_data.txt | ./main.exe

    一组测试结果已经存储在result.txt中

4. 使用小规模数据测试 (手动)

如果你需要调试特定的逻辑（如删除节点或查找失败的情况），建议进行小规模测试：

1.  直接运行测试：
    直接运行 `./main.exe`，然后在控制台手动输入数据。
    
2.  手动修改文件：
    打开 `test_data.txt`，手动修改其中的 N 值和节点数据，保存后按照第 3 节的方法再次运行。


5. 常见问题排查
终端编码：如果中文乱码，建议将终端编码设置为 UTF-8 (执行指令: chcp 65001)。
